# データシート・図面

設計で使った寸法の根拠となる、メーカー公式の図面・データシート。

## なぜリポジトリに置くか

`catalog.py` や `bom.yaml` に書いた寸法が「どこから来たか」を後から検証できるようにするため。
URL はリンク切れするが、ここに現物があれば根拠をたどれる。

**新しい部品を調べたら、PDF をここに保存し、下表に「何が読めるか」を追記すること。**
寸法の出所が不明な値を設計に使わない、というルールを支える仕組み。

## 収録物

### xl330/XL_XC-330.pdf

ROBOTIS DYNAMIXEL XL330 / XC330 シリーズの機械図面（図面日付 28-May-20、[FOR REFERENCE ONLY]）。

- 出典: https://emanual.robotis.com/docs/en/dxl/x/xl330-m288/#drawings
- 取得: 2026-07（ユーザーがブラウザで取得。robotis.com は CI/sandbox から到達不可）

読み取れる寸法:

| 項目 | 値 | 用途 |
|---|---|---|
| ケース外形 | 20 (W) x 34 (H) x 26 (D) mm | 干渉確認 |
| **出力軸の位置** | W方向は中央、H方向は**上端から 9.5 mm** | 軸基準の設計。箱の中央ではない |
| ホーン穴 | 4 x φ1.6、**P.C.D φ12**、90度等配 | 変換ハブの M2 締結 |
| ねじ込み深さ | DP 3.0 mm (Max) | M2x6 を使う（M2x8 は内部を破損） |
| IDLER 側 | 4 x φ1.6、P.C.D φ12、外形 φ16 | 反対側の支持 |

反映先: `hwlib/parts/xl330.py`、`hwlib/catalog.py`

### tamiya_70145/70145.pdf

タミヤ 楽しい工作シリーズ No.145 ナロータイヤセット（Item 70145）組立説明図。

- 出典: https://www.tamiya.com/japan/products/70145/index.html
- 取得: 2026-07

読み取れる寸法:

| 項目 | 値 | 用途 |
|---|---|---|
| タイヤ外径 | 58 mm | 走行速度・接地の計算 |
| タイヤ幅 | 16 mm | 軸方向の配置 |
| リム外径 | 42 mm | 変換ハブの外径スピゴット基準 |
| **ハブ取付穴** | 3 穴、**PCD 20 mm**（図面「10mm」= 軸中心から穴中心の半径 x2） | 変換ハブの3点締結 |
| 締結ビス | 3 x 8 mm タッピングビス | ハブへのねじ込み |

図面に数値が無い項目（実測が必要）:

- 中央ボア径、ハブ座ぐりの深さ → **本設計では中央ボアを使わない方針のため不要**
- ビス穴部の壁厚 → `wheel_boss_wall` として provisional に残っている

反映先: `hwlib/parts/tamiya_wheel.py`、`hwlib/catalog.py`
